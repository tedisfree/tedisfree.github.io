현재 작업중이던 도커 컨테이너를 다른 곳에서도 사용하기 위해 이미지화 하는 방법을 설명한다.

## 실행중이던 컨테이너를 이미지화 

```sh
docker commit -p [컨테이너ID or 컨테이너명] [생성할 이미지이름]
# -p : 이미지 생성중에 컨테이너를 일시정지
```

## 이미지를 압축(tar) 백업 
```sh
docker save -o <파일명.tar> <이미지 이름>:<태그>
# 태그는 임의의 값을 써 넣어도 된다.
```

## 압축파일로부터 복원 
```sh
docker load -i [백업한 파일이름].tar 
```
